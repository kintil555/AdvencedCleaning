# Advanced Cleaning

> Lightweight Windows system cleaner and optimizer — built with Electron and styled with [OreUI](https://github.com/Spectrollay-OreUI/OreUI).

![Build Status](https://github.com/kintil555/AdvencedCleaning/actions/workflows/build.yml/badge.svg)

---

## Features

| Feature | Description |
|---|---|
| 🧹 Temp Files | Clean Windows & user temp directories |
| 🌐 Browser Cache | Chrome, Edge, Firefox cache cleanup |
| 🗑 Recycle Bin | Empty all deleted files |
| ⚡ Prefetch | Clear Windows prefetch files |
| 📋 Windows Logs | Clear event logs & system logs |
| 🖼 Thumbnail Cache | Remove Explorer thumbnail databases |
| 🔄 Update Cache | Clean Windows Update download cache |
| 💥 Memory Dumps | Remove crash dump files |
| 🌐 DNS Flush | Clear DNS resolver cache |
| 🧠 RAM Optimize | Empty process working sets |
| 🚀 Startup Manager | View & remove startup registry entries |
| 🛡 SFC Scan | System File Checker |
| 🔬 DISM Repair | Windows component store repair |
| 🌐 Network Reset | Winsock + IP stack reset |

## Download

Go to [Releases](https://github.com/kintil555/AdvencedCleaning/releases) and download:
- `Advanced Cleaning Setup x.x.x.exe` — Installer
- `AdvancedCleaning-x.x.x-portable.exe` — Portable (no install needed)

> ⚠ **Run as Administrator** for full functionality (some cleaners require elevated rights).

## Build from Source

```bash
git clone https://github.com/kintil555/AdvencedCleaning
cd AdvencedCleaning
npm install
npm run build
```

Output will be in the `dist/` folder.
